## 📥 Быстрый старт

1. **[Releases](./releases/latest)** → скачайте zip.  
2. Распакуйте, откройте **`start.bat`** или **`download.bat`**.  
3. Вставьте ссылку на ролик → Enter → файл в папке `output/`.

```text
Пример (в bat уже прописано):
  yt-dlp "URL" -f "bestvideo+bestaudio/best"
```
